# Инструкция по добавлению фонового изображения

## ⚠️ ВАЖНО: Добавьте изображение перед запуском!

## Шаги для добавления фонового изображения:

1. **Поместите изображение в папку `public`:**
   - Скопируйте ваше изображение с фруктами и овощами
   - Переименуйте его в `background-food.jpg` (или используйте другое имя)
   - Поместите в папку `nutriplanner-frontend/public/`
   
   **Путь к файлу должен быть:** `nutriplanner-frontend/public/background-food.jpg`

2. **Альтернативный вариант (если хотите использовать другое имя):**
   - Если ваше изображение имеет другое имя, откройте файл `src/App.js`
   - Найдите строку: `backgroundImage: 'url(/background-food.jpg)'`
   - Замените `background-food.jpg` на имя вашего файла

3. **Формат изображения:**
   - Рекомендуется: JPG или PNG
   - Рекомендуемый размер: 1920x1080 или больше (для лучшего качества)
   - Оптимальный размер файла: до 2MB (для быстрой загрузки)

4. **После добавления:**
   - Перезапустите приложение (`npm start`)
   - Фоновое изображение будет отображаться на всех страницах
   - Элементы будут полупрозрачными с эффектом размытия (backdrop-filter)

## Текущие настройки:

- **Прозрачность фона:** 40% (opacity: 0.4)
- **Прозрачность элементов:** 85% (rgba(255, 255, 255, 0.85))
- **Эффект размытия:** blur(20px) saturate(180%)
- **Градиентное затемнение:** для лучшей читаемости текста

## Настройка прозрачности:

Если хотите изменить прозрачность элементов, откройте файл `src/index.css` и найдите:

```css
.transparent-card,
.MuiPaper-root,
.MuiCard-root {
  background: rgba(255, 255, 255, 0.85) !important;
  /* Измените 0.85 на нужное значение (0.0 - полностью прозрачно, 1.0 - непрозрачно) */
}
```

Для изменения прозрачности фонового изображения откройте `src/App.js` и найдите:

```javascript
opacity: 0.4  // Измените на нужное значение (0.0 - 1.0)
```
